# Unity Quest System

A comprehensive quest system package for Unity games.

## Installation

### Via Package Manager (Git URL)
1. Open Unity Package Manager
2. Click the "+" button
3. Select "Add package from git URL"
4. Enter: `https://github.com/joaovictoralencar/unity-quest-system.git`

### Via Package Manager (Local)
1. Download or clone this repository
2. Open Unity Package Manager
3. Click the "+" button
4. Select "Add package from disk"
5. Navigate to the package folder and select `package.json`

## Features

- Quest creation and management
- Objective tracking
- Reward system
- Event-driven architecture
- Editor tools for quest creation
- Sample scenes and examples

## Quick Start

1. Add the QuestManager prefab to your scene
2. Create quest assets using the Quest Creator tool
3. Implement quest objectives in your game logic
4. Use the provided UI components or create your own
